# YouTube数据提取工具

这是一个基于Python的YouTube数据提取和分析工具。

## 功能特点
- YouTube数据自动提取
- 数据可视化展示
- 在线数据查询

## 技术栈
- Python
- Flask Web框架
- YouTube Data API

## 部署信息
- 网站地址: https://y.cheesecatool.com/sj
- 部署平台: GitHub Pages
- CDN加速: Cloudflare

## 使用说明
1. 安装依赖: `pip install -r requirements.txt`
2. 运行应用: `python app.py`
3. 访问地址: `http://localhost:5000`
